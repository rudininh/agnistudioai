# Product Roadmap

Q1 membuktikan workflow konten individu. Q2 menambahkan analytics ingestion dan recommendation loop. Q3 mengaktifkan automation serta integrasi publishing terkontrol. Q4 memperluas kolaborasi workspace dan membuka capability platform untuk produk Agni lain.

Urutan ini dipilih karena data analytics dan automation hanya bernilai setelah pengguna memiliki workflow konten yang stabil.
