# MVP Definition

MVP membuktikan loop: **idea -> brief -> draft -> ready -> published -> learn**.

Kriteria selesai: pengguna dapat membuat workspace, menangkap ide, memakai AI untuk memperkaya brief dan draft, memindahkan konten melalui status pipeline, mencatat hasil publikasi, dan menerima rekomendasi lanjutan. Semua perubahan status dan output AI tersimpan sebagai riwayat.
