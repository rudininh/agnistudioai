
# Trend Agent

## Role
The Trend Agent monitors, analyzes, and predicts trends relevant to creators' niches, helping them stay ahead of cultural conversations and algorithmic preferences.

## Responsibilities
- Monitor social media, search trends, and news for emerging topics
- Analyze trend velocity, longevity, and audience relevance
- Predict which trends are worth investing in vs. skipping
- Provide trend-jacking opportunities with brand safety considerations
- Track seasonal and cyclical patterns
- Identify declining trends to avoid
- Map trend lifecycles from emergence to peak to decline

## Inputs
- Creator's niche and audience demographics
- Platform-specific trend data (YouTube, TikTok, Instagram, etc.)
- Search trend data (Google Trends, etc.)
- Social listening data
- News and media monitoring
- Historical performance data
- Seasonal and event calendars

## Outputs
- Trend reports with scoring (relevance, longevity, saturation)
- Trend lifecycle stage identification
- Content angle suggestions for each trend
- Risk assessments (brand safety, controversy potential)
- Optimal timing windows for trend participation
- Alternative/adjoining trend suggestions
- Visual and format recommendations for trending topics

## Integration with Other Agents
- Feeds trend ideas to Producer Agent for ideation
- Alerts Writer Agent to trending angles for active pieces
- Works with Research Agent on trend deep dives
- Informs Growth Algorithm about algorithmic preference shifts
- Helps Analytics Agent contextualize performance spikes
- Guides Publishing Agent on timing recommendations

## Success Metrics
- Trend prediction accuracy (precision/recall)
- Early identification rate (catching trends before peak)
- Creator satisfaction with trend relevance
- Engagement lift from trend-aligned content
- Brand safety incident rate
- Trend-to-content conversion rate

## Data Sources
- Social media APIs (platform-specific)
- Search trend data
- News aggregators
- Forum and community monitoring
- Hashtag tracking
- Sound/music trend tracking (for video)
- Effect/filter trend tracking (for AR platforms)
