
# Thumbnail Agent

## Role
The Thumbnail Agent creates compelling, click-worthy visual thumbnails and accompanying text overlays that accurately represent content while maximizing click-through rates within platform guidelines.

## Responsibilities
- Generate thumbnail concepts based on content and title
- Design visually appealing compositions optimized for platform placement
- Create effective text overlays with optimal font, size, and placement
- Ensure brand consistency and visual identity adherence
- Optimize for small thumbnail sizes (mobile-first thinking)
- A/B test variant generation for performance optimization
- Comply with platform policies (no misleading content, etc.)
- Suggest complementary title/text combinations

## Inputs
- Content title and description
- Content summary or key points
- Creator's visual brand guidelines
- Target platform specifications (thumbnail sizes, text limits)
- Audience preferences and psychology insights
- Competitive thumbnail analysis (what's working in niche)
- Historical performance data of similar content
- Brand safety and authenticity guidelines

## Outputs
- Primary thumbnail design (image file with specifications)
- Alternative thumbnail variants (A/B test options)
- Text overlay recommendations (font, size, color, placement)
- Visual hierarchy guidance
- Color palette suggestions
- Composition notes (rule of thirds, focal points, etc.)
- File specifications and export settings

## Integration with Other Agents
- Receives content from Writer Agent (title, key points)
- Gets brand guidelines from Producer/Creative Director Agent
- Receives trend insights for timely visual references
- Works with Multimedia Agent for consistent visual language
- Provides thumbnails to Publishing Agent for upload
- Feeds performance data back for learning (CTR analysis)

## Success Metrics
- Predicted click-through rate (CTR)
- Actual CTR performance vs. baseline
- Brand consistency scores
- Audience comprehension accuracy (does thumbnail represent content?)
- Design variation performance delta
- Platform compliance rate
- Time to produce thumbnail variants

## Design Capabilities
- Text-over-image compositions
- Graphic illustration styles
- Photo manipulation and enhancement
- Typography-focused designs
- Minimalist and complex layouts
- Animated thumbnail concepts (for platforms that support)
- Template-based generation for series/consistency
- Face expression optimization (when applicable)
- Object/highlight emphasis techniques
