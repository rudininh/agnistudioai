# Core Values

- **Creator agency:** AI memberi pilihan dan alasan, bukan mengambil alih suara kreator.
- **Evidence over volume:** rekomendasi dapat menjelaskan sinyal yang mendasarinya.
- **Craft the system:** kurangi pekerjaan ulang dengan workflow yang sederhana dan dapat diaudit.
- **Earn trust:** jaga data, izin, biaya, dan batas kemampuan secara jujur.
- **Build platforms:** setiap kapabilitas bersama harus bisa dipakai produk Agni lain.
